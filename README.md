# Aegis Agentic OS v2

Animated, clickable Agentic AI Operating System demo with FastAPI, WebSocket live workflow, upload panel, memory panel, and simulated AI Loop Engineering.

## Run on Windows PowerShell

```powershell
cd C:\Users\Christian\Downloads\aegis-agentic-os-v2\aegis-agentic-os-v2
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m uvicorn app:app --reload
```

Open:

```text
http://127.0.0.1:8000
```

## Features

- Animated dashboard
- Clickable sidebar and cards
- Live AI agent workflow
- WebSocket streaming
- AI Loop Engineering simulation
- Quality score animation
- File upload
- Memory center
- Settings placeholders
